# Dive Buddy · 水肺版(DiveScuba)安装指导

> 适用机型:**Garmin Descent G2**(其他机型请先联系卖家确认)
> 安装方式:侧载(拷一个文件进手表),全程约 5 分钟,不需要任何技术基础。

## 第 1 步:拿到表盘文件

购买/获取后你会收到一个文件:`DiveScuba.prg`。先把它保存到电脑上(微信/网盘传到电脑都行)。

## 第 2 步:把文件拷进手表

用**手表原装充电线**把手表连到电脑。

**Mac 电脑**(需要先装一个免费小工具 OpenMTP,Mac 默认看不到手表存储):
1. 打开 https://openmtp.ganeshrvel.com 点 Download,下载 dmg 安装(拖进 Applications 即可)
2. 打开 OpenMTP,右侧会出现手表的存储(看不到就换原装线、重插一次)
3. 进入 `GARMIN` → `APPS` 文件夹
4. 把 `DiveScuba.prg` 拖进 `APPS` 里
5. 拔线

**Windows 电脑**(免装任何软件):
1. 打开"此电脑",双击出现的 Garmin 设备
2. 进入 `GARMIN` → `APPS` 文件夹
3. 把 `DiveScuba.prg` 复制进去,拔线

## 第 3 步:表上启用表盘

拔线后等几秒(手表会自动识别新表盘):
1. 在表盘界面**长按中键(MENU)** → 选「表盘」
2. 翻到新出现的 **DiveScuba**,选中即可

> 找不到新表盘?拔线后等 10 秒再进一次列表;还没有就重启一次手表。

## 第 4 步:激活(配对码,30 秒)

首次使用,表盘会显示 **PAIR CODE + 6 位数字** 和网址。

1. 手机浏览器打开:**watch.xiaohuiwangai.cn/dive**,按页面上的 1-2-3 步进指引走
2. 在第 2 步输入表上的 6 位配对码——自动为你创建专属账户(第 1 步的 token 同时生成)
3. 第 3 步绑定佳明账号(在账户页完成):选择**佳明中国区**(garmin.cn 注册的选这个)或国际区,输入佳明账号密码
   - 密码只用于一次性登录你的佳明,**不会被保存**;开了两步验证的会再让你补一次验证码
4. 完成后**不用管手表**:表盘会在 10 分钟内自动完成激活(配对码消失),约 20-30 分钟内潜水数据上屏

## 常见问题

| 现象 | 原因/办法 |
|---|---|
| 配对码一直不消失 | 表盘每 10 分钟检查一次,输完码耐心等下一轮 |
| 配对码过期(1 小时) | 重新进一次表盘,会拿到新码,重输即可 |
| OpenMTP 看不到手表 | 换原装线;拔插一次;确认手表没在充电基座模式 |
| 数据显示 "--" | 绑定后首次拉取最长 30 分钟;确认佳明 App 里能看到你的潜水记录 |
| 手表数据多久更新 | 潜完水 10-20 分钟内自动上表;睡眠醒后 1-2 小时内;急就在网页面板点「🔄 立即同步」(约 10 秒出数) |

有问题联系卖家,或在 GitHub 提 issue:https://github.com/xiaohuiwang-ai/dive-buddy
